# Rexom Lavalink Plugins

This directory stores Lavalink plugins. They will be automatically downloaded when Lavalink starts.

## Configured Plugins

The following plugins are configured in `application.yml`:

1. **LavaSrc** - Provides Spotify, Apple Music, Deezer support
2. **LavaSearch** - Enhanced search functionality  
3. **YouTube Plugin** - Better YouTube support

## Note

You don't need to manually download plugins. Lavalink v4 downloads them automatically from the configuration.
